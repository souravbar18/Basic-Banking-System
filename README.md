#Smart Bank Account System
This is a simple Java-based console application designed to demonstrate the core concepts of Object-Oriented Programming (OOP), specifically focusing on Encapsulation.

🚀 Key Features
Encapsulation: The account balance is kept private to ensure data security and prevent unauthorized direct access.

Controlled Access: Implemented Getter and Setter (Deposit/Withdraw) methods to handle data modifications with validation logic.

Error Handling: Used try-catch blocks to manage NumberFormatException and ensure robust user input.

Interactive Console: A user-friendly menu system for performing banking operations.

🛠 Tech Stack
Language: Java

Concepts Applied: OOP, Encapsulation, Exception Handling, Control Flow, Scanner Input.

📋 How to Run
Clone this repository or download the source code.

Open your terminal in the project directory.

Compile the code: javac Main.java SmartAccount.java

Run the application: java Main

💡 Learning Objective
This project is part of my journey to master Object-Oriented Programming in Java. The primary goal was to understand how to protect data and implement secure communication between classes using methods.
